
<파일 속성>

크기, 종류, 접근권한, 링크 수, 소유자 및 그룹, 수정 시간		

<파일>
일반 파일 -			//데이터를 갖고 있는 텍스트 혹은 이진 파일
디렉토리 d			//디렉토리 내의 파일들의 이름과 파일 정보를 관리하는 파일
문자 장치 파일 c		//문자 단위로 데이터를 전송하는 장치를 나타내는 파일
블록 장치 파일 b		//블록 단위로 데이터를 전송하는 장치를 나타내는 파일
FIFO 파일 p		//프로세스 간 통신에 사용되는 이름 있는 파이프
소켓 s			//네트워크를 통한 프로세스 간 통신에 사용되는 파일
심볼릭 링크 l		//다른 파일을 가리키는 포인터 역할을 하는 파일



file (대상)		//파일의 종류에 대한 자세한 정보를 출력한다.
	$ file cs1.txt
	cs1.txt: ASCII text



<접근권한>
Read, Write, Execute 권한이 존재함.

소유자/그룹/기타로 구분하여 관리
	ex) rwxr-xr--

<접근권한 변경>
chmod (대상)		//대상은 파일 혹은 디렉토리. -R 옵션: 디렉토리 하위 파일에도 적용한다.

::8진수 표현::

ex) rwxr-xr--: 111,101,100이므로 754.
$ chmod 754 temp.txt

rwxrwxrwx 777
rwxr-xr-x 755
rw-rw-r-- 664
rw-r--r-- 644
rwx------ 700
rw-rw-rw- 666

::기호 표현::

사용자 범위
User		소유자
Group		그룹
Others		기타 사용자
All		모든 사용자

연산자
+ 	추가
- 	제거
=	권한 설정

권한
r
w
x


ex)
$ chmod g-w temp.txt
$ chmod o-r temp.txt
$ chmod g+w,o+rw temp.txt




<기타 파일 속성 변경>

chown (대상)		//대상은 파일이나 디렉토리, -R으로 하위파일도 적용
chgrp (대상)		//동일	
touch (대상)		//최종수정시간 톡


<핵심>

링크는 기존의 파일에 대한 또 하나의 새로운 이름으로 하드 링크와 심볼릭(소프트) 링크가 있다.
파일은 이름뿐만 아니라 타입, 크기, 소유자, 접근권한, 수정 시간 등의 파일 속성을 갖는다.
파일의 접근권한은 소유자, 그룹, 기타로 구분하여 관리한다.
